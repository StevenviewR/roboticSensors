1 Installing Label Studio

  1. Search "cmd" from windows, run as administrator
    .

  2. Check installed python

python3 --version


download the python installer, if there is no python installed.
https://www.python.org/downloads/

  1. Navigate to your Documents folder
cd "%USERPROFILE%\Documents"
  1. Create a folder
mkdir yolo_projects && cd yolo_projects
  1. Create the virtual environment:
python3 -m venv venv39
  1. Activate the environment
cd venv39 && Scripts\activate

  1. Install Label-Studio
pip install label-studio
